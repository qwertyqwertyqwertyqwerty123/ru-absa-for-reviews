## Processed data

This folder contains processed and auxiliary dataset files used in the project.

### Files
- `dataset_stats.json` - summary statistics of the dataset, including aspect and sentiment distributions.
- `sample_data.jsonl` - a small sample of annotated records for demonstration and testing.
___
## Обработанные данные

В данной папке находятся обработанные и вспомогательные файлы датасета, используемые в проекте.

### Файлы
- `dataset_stats.json` — сводная статистика по датасету, включая распределение аспектов и тональностей.
- `sample_data.jsonl` — небольшой пример размеченных записей для демонстрации и тестирования.
