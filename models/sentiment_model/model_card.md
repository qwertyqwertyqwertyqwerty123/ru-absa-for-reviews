# Sentiment Classification Model

## Task

Predict sentiment for a review fragment and an aspect. Binary.

## Base model

`DeepPavlov/rubert-base-cased`

## Intended use

The model is intended for pet-project experiments with Russian ABSA pipelines.

## Limitations

- Not intended for production use without additional validation.
- Quality depends on aspect extraction quality and labeled sentiment pairs.
- Checkpoint weights are stored outside GitHub.
