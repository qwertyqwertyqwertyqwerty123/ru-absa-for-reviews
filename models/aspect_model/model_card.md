# Aspect Detection Model

## Task

Detect review aspects in Russian-language user reviews.

## Base model

`DeepPavlov/rubert-base-cased`

## Intended use

The model is intended for pet-project experiments with Russian ABSA pipelines.

## Limitations

- Not intended for production use without additional validation.
- Quality depends on source annotation quality and domain coverage.
- Checkpoint weights are stored locally.
